## 🛠️ Stack Tecnológico

- **Frontend:** NextJS 14 + TailwindCSS
- **Backend:** Node.js + Express
- **Banco de Dados:** SQLite / Supabase
- **Deploy:** Vercel (free tier)
- **Plantas:** Fabric.js + PDF.js
